## 팀 그라운드 룰

우리팀의 규칙

스크럼 주제
- 컨디션, 건강 상태 체크
- 오늘 할 것 정하기
- 어제 한 것 정리해보기

프로젝트 규칙
- fork: 제이콥이 포크

- 커밋메세지 카르마 스타일 준수
    - feat

    - docs
